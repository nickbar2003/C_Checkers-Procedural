# C Checkers Game

This is a remake of an older project.

I recreated the game Checkers using the C programming lanugage, and RayLib graphics library!

There are two new versions of this project. One using C and another in C++. The two versions use different approaches for implementing the same game.

The C version has a more data/procedural oriented approach,

whereas the C++ version is more OOP in nature. (You can find the C++ at https://github.com/nickbar2003/Cpp_Checkers-Object-Oriented)
